# KeySwitcher

Фоновый резидент для Windows, автоматически исправляющий слова, набранные в неверной
раскладке (аналог Punto Switcher), с поддержкой **русского, украинского и английского**.

## Возможности

- Глобальный перехват клавиатуры (`WH_KEYBOARD_LL`) и отслеживание набираемого слова.
- **Ручной режим:**
  - **Break** — переписать последнее/набираемое слово в наиболее подходящей раскладке
    и переключить раскладку.
  - **Shift+Break** — конвертировать выделенный текст (через буфер обмена).
- **Автозамена «на лету»** (переключается в меню трея; по умолчанию выключена):
  по пробелу при уверенной детекции слово исправляется автоматически.
- Гибридная детекция: словари RU/UA/EN + эвристика (алфавит, гласные, буквы-сигнатуры)
  + **биграммная n-gram модель**, которая подключается при загрузке полного словаря.
- **Исключения:** не трогает URL, e-mail и пути; не срабатывает в полях ввода пароля;
  внутри-токенные разделители (точка/слэш/@ в URL, числах, путях) не рвут токен.
- **Отмена последней замены** и **окно настроек** (авто-режим, автозапуск, набор языков,
  папка словарей) — из меню трея.
- **Автозапуск** при входе в систему (реестр `Run`).
- **Пользовательские словари:** положите полные частотные списки в
  `%AppData%\KeySwitcher\dict\{en,ru,uk}.txt` — они сольются со встроенными без пересборки
  (поддержан формат Hermit Dave `слово частота`).
- **Управление языками:** ненужный язык можно отключить в настройках.
- Значок в трее, единственный экземпляр, настройки в `%AppData%\KeySwitcher\settings.json`.

## Требования

- Windows 10/11
- .NET 8 SDK (для сборки) — `winget install Microsoft.DotNet.SDK.8`

## Сборка и запуск

```powershell
dotnet build KeySwitcher.sln -c Release
dotnet run --project src/KeySwitcher/KeySwitcher.csproj -c Release
```

Приложение стартует без окна — ищите значок в области уведомлений.

## Тесты

```powershell
dotnet test KeySwitcher.sln
```

## Ручная проверка (smoke-тест)

1. Запустите приложение → появится значок в трее.
2. В Блокноте наберите `ghbdtn`, нажмите **Break** → получится `привет`, раскладка → RU.
   Обратно: `hello` в русской раскладке (`руддщ`) + Break → `hello`, раскладка → EN.
3. Наберите `ghbdsn`, нажмите **Break** → `привіт` (украинский).
4. Выделите слово в неверной раскладке, нажмите **Shift+Break** → текст конвертируется,
   исходный буфер обмена восстанавливается.
5. Включите «Автозамена «на лету»» в меню трея и наберите `ghbdtn` + пробел →
   слово исправится автоматически.
6. Сразу после замены выберите «Отменить последнюю замену» → вернётся исходный ввод.
7. Откройте «Настройки…»: отключите ненужный язык, включите автозапуск, откройте папку
   словарей и положите туда полный `ru.txt` / `uk.txt` / `en.txt`.

## Структура

```
src/KeySwitcher/
  Program.cs                 entry, single-instance
  App/TrayAppContext.cs      трей + маршрутизация клавиш
  App/SettingsForm.cs        окно настроек
  Interop/                   WinAPI: хук, SendInput, раскладки, буфер обмена, VK→символ,
                             автозапуск (реестр), детекция поля пароля
  Core/                      WordBuffer, LayoutMap, LayoutConverter, LayoutDetector,
                             NgramModel, TextExclusions, ManualSwitcher
  Dictionaries/              WordDictionary + встроенные словари (resources/{ru,uk,en}.txt)
  Config/Settings.cs         настройки (JSON): авто-режим, набор языков
tests/KeySwitcher.Tests/     unit-тесты ядра (xUnit)
```
